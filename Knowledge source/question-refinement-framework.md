## 1. THE EXAMINER BLUEPRINT

Real FRCR 2B long cases follow a repeatable structural formula. They do NOT test spot diagnosis. They test **diagnostic reasoning through a clinical journey**.

### The 6-Step Case Arc

```
Step 1: ANCHOR ABNORMALITY
  - Not too obvious (not "lobar pneumonia")
  - Not too rare (not a zebra)
  - Forces pattern recognition + compartment reasoning
  - Examples: mediastinal widening, pleural-based opacity, multiple pulmonary nodules,
    solitary lytic bone lesion, adnexal mass, dilated bowel loop

Step 2: COMPARTMENT-BASED DIFFERENTIAL CLUSTERING
  - Examiners LOVE location-based reasoning
  - Force candidates into compartment thinking:
    * Anterior / middle / posterior mediastinum
    * Intra-axial / extra-axial
    * Intradural / extradural / intramedullary
    * Intraperitoneal / retroperitoneal
    * Pleural / parenchymal / chest wall
    * Lytic / sclerotic / mixed
    * Epiphyseal / metaphyseal / diaphyseal

Step 3: MODALITY ESCALATION (The Story)
  - Cases must tell a story across imaging modalities:
    * CXR → CT
    * Ultrasound → CT/MRI
    * CT → MRI
    * CXR → PET-CT
    * Plain film → MRI
    * CT pelvis → SPECT (nuclear medicine)
  - Each modality reveals more detail and increases complexity
  - The clinical history field should reference the initial modality findings

Step 4: CLINICAL INFLECTION POINT
  - Something in the clinical context shifts probability:
    * Age (child vs adult vs elderly)
    * Immunosuppression (HIV, transplant, chemotherapy)
    * Trauma history
    * Known primary malignancy
    * Constitutional symptoms (weight loss, night sweats)
    * Raised tumour markers (AFP, CEA, CA-125, beta-hCG)
    * TB risk factors (travel, immunocompromise)
    * Post-surgical/post-treatment history
    * OCP use, pregnancy/post-partum
    * Occupational exposure (asbestos, silica)

Step 5: FORCE PRIORITISATION
  - Not a list of differentials — a RANKED differential
  - Conditional differential approach: "If the patient is X, then Y is more likely"
  - Include discriminating features for each differential

Step 6: MANAGEMENT LAYER
  - Biopsy? (who performs it, what approach?)
  - Surgery? (is it resectable?)
  - Staging? (what further imaging?)
  - MDT discussion?
  - Urgent vs routine referral?
  - Specific specialist referral (oncology, neurosurgery, respiratory, etc.)
```

## 2. WHAT THE RADIOPAEDIA MODEL ANSWERS TEACH US

### Key Patterns from Real Scoring

1. **Multi-modality cases are standard** — the discitis case uses CXR + CT, the renal osteodystrophy case uses CT pelvis + Tc-99m SPECT, the masticator space case uses MRI + CT.

2. **Findings must be modality-separated** — when multiple modalities are shown, findings must be listed under each modality heading separately (e.g., "CXR: ...", "CT: ...").

3. **Important negatives earn marks** — examiners explicitly reward candidates who mention intact structures, absent complications, negative lymphadenopathy, etc.

4. **Incidental findings earn marks** — the gallbladder case awards marks for "extensive vascular calcification" as a bullet. The discitis case awards marks for "paraseptal emphysema" and "dilated pulmonary trunk".

5. **Comprehensive answers score higher** — examiners consistently comment that brief answers limit scores. The top-scoring answers describe ALL findings.

6. **The diagnosis section should include multiple components** — e.g., "T11/12 discitis osteomyelitis with phlegmon/abscess + right pleural empyema + multifocal lung infection/septic emboli" (not just "discitis").

7. **Management must be specific** — "orthopaedic referral" alone scores less than "urgent MRI spine to assess for epidural abscess/cord compression, investigate for endocarditis, pleural fluid aspiration".

### Examiner Feedback Themes (from real scored answers)

| Score | What Examiners Say |
|-------|-------------------|
| 4-5 (fail/borderline) | "Failed to recognise primary abnormality", "described simple cholecystitis only", "not considered sarcoma in differential" |
| 5.5-6 (pass) | "Correct diagnosis but concise answer limits marks", "missed secondary pathology", "did not describe additional features" |
| 6.5-7 (clear pass) | "Good answer, missed one component", "did not describe invasive nature", "mentioning intact structures would have resulted in higher score" |
| 7-7.5 (clear pass) | "Excellent answer", "described almost all features", "correct diagnosis + bilateral meniscal tears + 5/8 bullets" |
| 8 (top mark) | "Comprehensive answer with primary and secondary diagnoses, 7/8 key bullets, important negatives, and appropriate management" |

## 3. WHAT THE 100 CASES BOOK TEACHES US

### The Conditional Differential Framework

The book teaches a powerful technique: **conditional differential diagnosis using "If..."**

Instead of: "Differentials include A, B, C"
Use: "If the patient is acutely unwell, I would consider A. If the presentation has been more insidious, B is more likely. If there is a past history of malignancy, C should be excluded."

**This is the gold standard for long case differentials.**

### Case Presentation Structure (from the book)

1. **Open**: "This is a [modality] of a [age/sex] patient..."
2. **Describe obvious abnormality first** — don't waste time searching before speaking
3. **Describe systematically** — use a review pattern for each modality
4. **Summarise** — "So, we have..." or "To summarise..."
5. **Conditional differential** — "If... then..." approach
6. **Management** — "I would discuss this urgently with..." / "Further investigation with..."
7. **Close** — signal you're done and ready for dialogue

### Key Principles

- **Not spot diagnosis** — cases test perception + interpretation + management synthesis
- **Complex cases with multiple findings are common** — e.g., pneumoperitoneum + metastatic breast cancer + pathological fracture (Case 5 in the book)
- **Lines and tubes first** — always mention them if present
- **Lumps: benign vs aggressive first** — then refine differential
- **Age-appropriate differentials** — child vs adult vs elderly
- **Terminology matters** — density (plain film), attenuation (CT), signal intensity (MRI), reflectivity (US), tracer uptake (NM)

## 5. CLINICAL HISTORY DESIGN PRINCIPLES

The clinical history is NOT throwaway text. It is deliberately designed to:

### Must Include:
1. **Age and sex** — always shifts differential probability
2. **Presenting complaint** — the clinical trigger
3. **Duration** — acute vs chronic fundamentally changes differential
4. **One clinical inflection point** — the thing that shifts probability (see Step 4 above)
5. **Relevant investigation results** — e.g., "AFP 450", "ACE levels elevated", "D-dimer markedly elevated"

### Should Include (where relevant):
6. **Previous imaging reference** — "Surveillance US shows a 4cm lesion. MRI for characterisation."
7. **Referral context** — "Referred for staging prior to surgical planning"
8. **Risk factors** — "smoker", "HIV positive, CD4 80", "on long-term OCP", "asbestos exposure"

### Must NOT Include:
- The diagnosis in the clinical history
- Information that makes the diagnosis trivially obvious
- Irrelevant clinical detail that doesn't inform the case

### The Modality Escalation Pattern in Clinical History

**Good example** (from Radiopaedia — discitis case):
> Clinical history references CXR findings → CT chest shown → forces candidate to interpret both and synthesise

**Good example** (HCC case from our data):
> "Surveillance ultrasound shows a 4cm hypoechoic lesion in segment VI. MRI liver with Primovist for characterisation."

This naturally creates the story: US found something → MRI to characterise → candidate must interpret MRI in context of US findings.

## 6. FINDINGS FORMAT STANDARD

### From Radiopaedia Gold Standard:

```
Multi-modality cases: Separate findings by modality heading

CXR
- finding 1
- finding 2

CT
- finding 1
- finding 2
- finding 3

Single-modality cases: Direct bullet list
- finding 1
- finding 2
```

### Findings Must Include:
1. **Primary abnormality** (described first, most detail)
2. **Secondary/associated findings** (supporting the diagnosis)
3. **Complications** (if present)
4. **Incidental findings** (if clinically relevant — these earn marks!)
5. **Important normal structures** (at least 2 — "PCL intact", "no biliary dilatation")

### Language Rules:
- Terse radiological language — no full sentences
- Correct modality terminology (density/attenuation/signal/reflectivity/uptake)
- Describe what you SEE, not what you think it is (save interpretation for later)
- Quantify where possible ("4cm", "moderate volume", "multiple bilateral")

## 7. SCORING GUIDE PRINCIPLES

### From Radiopaedia Marking Scheme:

Each case must have a **case-specific** scoring guide. The generic template is useful as a starting point but the REAL value is in case-specific scoring.

**Pattern for scoring guides:**

| Score | What It Means |
|-------|---------------|
| 3 | No answer offered |
| 4 | Fails to identify the primary pathology entirely |
| 5 | Identifies primary pathology but with significant omissions OR correct diagnosis with wrong description |
| 6 | Correct primary diagnosis + adequate findings but misses secondary pathology or key feature |
| 7 | Correct primary + secondary diagnosis + 5/8 key bullets |
| 8 | Comprehensive: primary + secondary + 7/8 bullets + important negatives + management |

**Critical insight**: The difference between 6 and 7 is always about **secondary findings and key bullets**. The difference between 7 and 8 is about **important negatives and management comprehensiveness**.

## 8. CASE DESIGN QUALITY CHECKLIST

Before finalising any long case, verify:

- [ ] **Story arc**: Does the case tell a clinical story across modalities?
- [ ] **Not a spot diagnosis**: Does it require synthesis, not just recognition?
- [ ] **Compartment reasoning**: Does it force anatomical compartment thinking?
- [ ] **Clinical inflection point**: Is there something in the history that shifts probability?
- [ ] **Differential cluster**: Does it map to a known high-yield examiner cluster?
- [ ] **Multiple components**: Does the diagnosis have >1 component? (e.g., "discitis + empyema + septic emboli")
- [ ] **Incidental findings**: Are there 1-2 incidental findings that earn marks?
- [ ] **Important negatives**: Are there 5 things the examiner expects you to actively exclude?
- [ ] **Management depth**: Is the management specific (not just "refer to specialist")?
- [ ] **Age-appropriate**: Are differentials appropriate for the stated age?
- [ ] **UK practice**: Does management reflect UK radiology practice?
- [ ] **Modality terminology**: Is correct terminology used for each modality?
- [ ] **Mnemonic alignment**: If a mnemonic exists for this case's section (see Section 11), are differentials consistent with the mnemonic's differential list?
- [ ] **Mnemonic exam pearl**: If a relevant mnemonic exists, does the `examPearl` reference it?

## 9. CASE ARCHETYPE TEMPLATES

### Archetype A: Multi-Modality Escalation
```
Clinical History: [Initial modality] shows [anchor finding]. [Cross-sectional] performed for further evaluation.
Pattern: Plain film/US → CT/MRI
Example: CXR shows mediastinal widening → CT reveals anterior mediastinal mass
```

### Archetype B: Staging Journey
```
Clinical History: Known [primary malignancy]. Referred for [staging/follow-up].
Pattern: Biopsy-proven diagnosis → Cross-sectional staging
Example: Biopsy-proven SCC right upper lobe → PET-CT for staging
```

### Archetype C: Incidental Finding Characterisation
```
Clinical History: Incidental [finding] on [initial modality]. [Characterisation modality] performed.
Pattern: Incidental US/CT finding → Dedicated MRI characterisation
Example: Incidental liver lesion on US → MRI with Primovist for characterisation
```

### Archetype D: Acute Presentation with Complexity
```
Clinical History: [Acute presentation] with [clinical signs]. [Emergency imaging].
Pattern: Emergency clinical scenario → Emergency cross-sectional
Example: Acute abdomen + known Crohn's → CT abdomen/pelvis
```

### Archetype E: Multi-System Disease
```
Clinical History: [Systemic symptoms] affecting multiple systems. [Appropriate imaging].
Pattern: Constitutional symptoms → Imaging reveals multi-system involvement
Example: TB — CXR + spinal involvement → CT shows discitis + empyema + septic emboli
```

### Archetype F: Nuclear Medicine Correlation
```
Clinical History: [Clinical context]. [NM study] performed with [CT correlation].
Pattern: Nuclear medicine + anatomical correlation
Example: Renal osteodystrophy — CT pelvis + Tc-99m sestamibi SPECT
```

## 10. ANTI-PATTERNS (What NOT to Do)

### Weak Case Design:
- "CXR shows widened mediastinum. What is the diagnosis?" — too flat, no story
- Single finding, single modality, no clinical context
- Diagnosis is obvious from the clinical history alone
- Differentials that don't require compartment reasoning
- Management that is generic ("refer to specialist")

### Weak Findings:
- Prose paragraphs instead of bullet points
- Missing important negatives
- No incidental findings
- Findings that don't match the stated modality terminology
- Findings that give away the diagnosis too obviously in the first bullet

### Weak Scoring:
- Generic scoring guide copied across all cases
- No case-specific scoring criteria
- Bullets that don't discriminate between pass and fail candidates

## 11. MNEMONIC INTEGRATION

> Source: "Mnemonics for Radiologists and FRCR 2B Viva Preparation" — 50 chapters, each with a mnemonic covering a high-yield differential pattern.

### Purpose

Mnemonics provide a structured, memorable framework for differential diagnosis. When populating long cases, differentials MUST align with the relevant mnemonic where one exists. This ensures:
1. Candidates learn differentials in a structured, reproducible way
2. Cases are internally consistent with the mnemonic seed data
3. Exam pearls can reference mnemonics as high-yield revision aids

### Rules for Mnemonic Integration

1. **Differential alignment**: When a long case's section maps to a mnemonic chapter, the case's differentials MUST include conditions from the mnemonic's differential list. It is acceptable to narrow the list (e.g., pick the 3 most relevant for that specific case), but they should not contradict or ignore the mnemonic.

2. **Exam pearl references**: Each case's `examPearl` SHOULD reference the relevant mnemonic where one exists. Format: *"The mnemonic **MAGIC DR** is useful for ring-enhancing brain lesions: Metastasis, Abscess, Glioblastoma, Infarct, Contusion, Demyelination, Radiation necrosis [n]."*

3. **Conditional differentials from mnemonics**: When using the "If... then..." conditional differential approach (Section 3), the mnemonic's conditions should inform the conditional branches. For example: *"If immunocompromised, MAGIC DR directs us to Abscess or Lymphoma as the leading differentials."*

4. **No mnemonic required**: If a case's section has no corresponding mnemonic chapter, this rule does not apply. Not all sections have mnemonics.

1. **`differentialPatterns` table** (Convex) — 325 records

   - Empty shells: findings, interpretation, nextSteps, keyBullets, importantNegatives, examPearl

### Integration Rules
1. When populating a long case whose pattern matches an O'Brien differential, the case's differentials SHOULD include the O'Brien top-3 diagnoses
2. The `differentialPatterns` table serves as a cross-reference: query by section to find all relevant differential patterns for a given case type
3. O'Brien case numbers map 1:1 with `differentialPatterns.obrienCaseNumber`
4. Each O'Brien pattern is tagged with `textbookRefs: [7]`

### Pipeline Usage
- **Case generation**: Use `differentialPatterns.getBySection(section)` to find all relevant top-3 lists when writing a new long case
- **Quality validation**: Check that case differentials align with O'Brien top-3 when a matching pattern exists
- **Mnemonic cross-reference**: O'Brien patterns overlap with mnemonic chapters — e.g., O'Brien case 1 (Solitary pulmonary nodule) relates to mnemonic chapters 4 (CAVITIES)
- **Viva prep**: Top-3 differential lists are ideal for viva practice ("What are the top 3 differentials for this pattern?")
