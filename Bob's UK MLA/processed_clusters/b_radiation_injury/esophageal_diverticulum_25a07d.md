# Finding: Esophageal Diverticulum

**Source section:** B. RADIATION INJURY
**Quality score:** 0.5
**Differential count:** 6

## Differentials

- Zenker diverticulum (pharyngoesophageal)
- Killian-Jamieson diverticulum
- Interbronchial diverticulum =  traction diverticulum =  response to pull from fibrous adhesions following lymph node infection (TB), contains all 3 esophageal layers
- Interaorticobronchial diverticulum =  thoracic pulsion diverticulum Location:   on left anterolateral wall between inferior border of aortic arch + upper margin of left main bronchus
- Epiphrenic diverticulum (rare)
- Intramural esophageal pseudodiverticulosis

## Context
