# Finding: PULSATILITY

**Source section:** PHASICITY
**Quality score:** 0.467
**Differential count:** 5

## Differentials

- Full pulsatility index of Gosling (PIF) = 1/A02 SAi2
- Simplified pulsatility index (PI) = (S - D)/mean
- Resistance index (RI) = Pourcelot index = (S - D)/S or 1 - (D/S)
- Stuart index = A/B ratio = S/D ratio
- B/A ratio = B(100%)/A

## Context

=  assessment of vascular resistance (increased resistance reduces diastolic flow) ◊  Can be assessed in vessels too small / tortuous to be imaged (Doppler angle unnecessary)! ◊  Index should be calculated for each of several cardiac cycles (5 heartbeats adequate) an average value taken S = A = maximal systolic shift D = B = end-diastolic frequency shift
